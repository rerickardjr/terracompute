# TerraCompute

TerraCompute is a platform designed to reconcile the expansion of High-Performance AI computing with ecological and community stewardship. 

We develop software and structural frameworks to:
1. **Optimize AI Workload Schedulers** for carbon-aware, grid-stabilizing, and water-free operations.
2. **Accelerate Siting Decisions** through GIS mapping of non-arable lands, industrial brownfields, and bedrock-encased cooling environments.
3. **Engage Communities** via transparent, real-time public telemetry dashboards that verify ratepayer protection and ecological alignment.

## Project Structure
- `/assets`: Brand design, logos, and UI wireframes.
- `/services`: Core workload scheduler prototypes and GIS mapping configurations.
- `/dashboard`: Prototype for the public-facing community impact dashboard.
